## 파이썬 특징

### 인터프린터 언어
직관적임, 파이썬은 사람이 생각하는 방식을 그대로 표현할 수 있는 언어임 따라서 프로그래머는 굳이 컴퓨터의 사고 체계에 맞추어서 프로그래밍을 하려고 애쓸 필요가 없음

### 간단하고 쉬운 문법
비전공자들도 배우기 쉬움

### 고수준의 내장 객체 자료형 제공
유연한 개발 가능

### 메모리 자동관리
Garbage Collection기능을 사용하여 메모리를 자동 할당함

### 높은 확장성
다른 언어와 쉽게 결합시킬 수 있음
